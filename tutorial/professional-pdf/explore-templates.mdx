Now that you know the workflow — initialise a template, describe what you want, compile, review — let's explore what else you can create. Each template below comes from [Typst Universe](https://typst.app/universe), Typst's free community template library.

```mermaid
flowchart LR
    A["You speak\nor type"] -->|"Wispr Flow\n(optional)"| B["Gemini CLI\ncustomises it"]
    B -->|"compile"| C["Typst creates\nyour PDF"]
    C -->|"review"| D["Use or\nrefine it"]
```

Pick any template below. Run the `typst init` command to download it, then use the prompts to customise it — say them out loud with Wispr Flow or copy and paste.

  <Accordion title="1. Cover Letter — Advanced (Simplest)">
    **Use case:** Take your cover letter further with multi-page or styled variants.

    You already customised the fireside template — here's how to level it up.

    Download the fireside template by running this command:

    ```bash title="Copy this command"
    typst init @preview/fireside:1.0.0
    ```

    Start Gemini CLI in the template folder, then say or copy this prompt:

    ```text title="Say this or copy this prompt"
    I have a fireside cover letter template in this folder.
    Please customise it into a visually distinctive cover letter:
    - Add a professional header with my name, email, phone, and LinkedIn URL
    - Add a sidebar or accent column with a personal brand colour
    - Today's date in NZ format (e.g. 19 March 2026)
    - A well-structured body with clear paragraphs
    - A professional sign-off
    Use placeholder content. Use NZ English spelling.
    Make it stand out from a standard letter.
    Then compile it to PDF.
    ```

    **Follow-up customisations:**

    ```text title="Say this or copy this prompt"
    Add a matching header and footer with a thin coloured line. Put my name
    in the header and page number in the footer. Then recompile.
    ```

    ```text title="Say this or copy this prompt"
    Extend this into a two-page version with an additional section for
    "Key Achievements" with 3-4 bullet points. Keep the same design style.
    Then recompile.
    ```

    <Tip>
    **NZ job market tip:** Many NZ employers still appreciate a well-formatted cover letter. A visually distinctive design can help your application stand out — especially for roles in marketing, design, or communications.
    </Tip>
  </Accordion>

## Design from Scratch

Once you're comfortable with templates, you can skip `typst init` entirely and ask Gemini CLI to create documents from a blank page:

```text title="Say this or copy this prompt"
Create a [document type] as a Typst file called [filename].typ.
Don't use any template — design it from scratch with:
- [describe your layout and content requirements]
Use NZ English spelling. Then compile it to PDF.
```

This gives you complete creative freedom — but templates are usually a faster starting point.

## Find More Templates

The [Typst Universe](https://typst.app/universe) has hundreds of free templates. Browse it for inspiration, then use `typst init @preview/template-name:version` to get started.

<Tip>
**Mix and match!** You can combine elements from different templates. For example, say: "Take the header style from my cover letter and use it on my invoice" — Gemini CLI can handle this because all your `.typ` files are in the same folder.
</Tip>

# HER WAKA Documentation — Project Instructions

## About this project

- This is the documentation site for **HER WAKA**, an employment-focused programme delivered by SheSharp for MSD clients in Auckland (March–June 2026)
- Built on [Mintlify](https://mintlify.com) with the "almond" theme
- Pages are MDX files with YAML frontmatter
- Configuration lives in `docs.json`
- Run `npx mint dev` to preview locally
- Run `npx mint broken-links` to check links

## Terminology

- Use **"participant"** not "student" or "attendee"
- Use **"SheSharp"** (camelCase, capital S and S) — never "She Sharp" or "shesharp"
- Use **"HER WAKA"** (all caps) — never "Her Waka" or "her waka"
- Use **"programme"** (NZ English spelling) not "program"
- Use **"Aotearoa New Zealand"** or **"New Zealand"** — not "NZ" in prose (abbreviation OK in tables)
- Use **"facilitator"** not "teacher" or "trainer"
- Use **"session"** not "class" or "lesson"

## Style preferences

- Use active voice and second person ("you")
- Keep sentences concise — one idea per sentence
- Use sentence case for headings
- Bold for UI elements: Click **Settings**
- Code formatting for file names, commands, paths, and code references
- Use New Zealand English spelling (e.g., "organisation", "colour", "recognise")
- Tone: warm, encouraging, professional — participants may be new to tech

## Content boundaries

- Do not document internal SheSharp operations or financials
- Do not include participant personal information
- Keep MSD references general — link to official MSD resources rather than reproducing policy
- Workshop slide content should be summarised, not reproduced in full

- Dark mode accent: `#9b2e83` (Purple Dark)
